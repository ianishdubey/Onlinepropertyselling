🏡 Online Property Selling Platform
Repository: ianishdubey/Onlinepropertyselling
Tech Stack: Pure HTML, CSS, and JavaScript (Frontend)
Role: Full-stack developer for frontend components; crafted user and admin interfaces

🔍 Project Overview
An intuitive, responsive website designed to streamline the property browsing and selling process:

User interface for browsing properties, viewing details, and submitting inquiries to sellers

Admin dashboard for managing property listings—addition, modification, and deletion of entries

Fully built with HTML, CSS, and vanilla JavaScript, ensuring compatibility and maintainability

⚙️ Key Features
For End Users:

Clean, responsive layout showcasing properties with images, specs, and descriptions

Contact form integration that allows direct communication with sellers

For Admins:

Secure admin panel (controlled via JavaScript logic) for CRUD operations on listings

Intuitive controls to update or remove listings easily

🛠 Tech Highlights
Frontend-first approach: No back-end reliance; all interactions simulated via structured JavaScript

Dynamic UI updates: JavaScript-driven DOM manipulation renders real-time changes for both user and admin workflows

Responsive design: CSS media queries ensure seamless viewing on mobile, tablet, and desktop screens
